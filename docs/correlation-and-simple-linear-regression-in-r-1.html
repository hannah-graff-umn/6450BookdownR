Software Lesson 13 Correlation and Simple Linear Regression in R

13.1 Goal of Software Lesson 13

The goal of this software lesson is to learn how to

  • obtain test statistics, -values, and confidence intervals for the regression coefficients, and
  • create diagnostic plots (e.g., residual plot and QQ plot) for the regression model.


Let’s use the Lung Health Study (LHS) data set again.

Recall that the data set and a description of data can be found on Canvas, titled lhs.csv and lhsdoc.docx, respectively.

lhs<-read.csv(file = file.choose(), header = TRUE)

Last week, you investigated the relationship between two numerical variables via a graphical display (i.e., scatterplots) and summary measures (i.e., correlation, regression coefficients). This week, you are going to build on this knowledge and learn how to obtain inferential results (e.g., CI’s, p-values) and diagnostic information for simple linear regression (SLR).

You will be investigating the same scenario as in the previous lesson; that is, the relationship between the number of cigarettes smoked per day at baseline (f10cigs) and FEV1/FVC\(\%\) at baseline (FEVFVC02).

13.1.1 Inference for Simple Linear Regression

Last week, you obtained regression coefficients (i.e., intercept and slope) from the SLR model to predict Y from X. Recall that the code to obtain these values also provided you with inferential results (i.e., p-values) regarding the regression coefficients. This code and explanation are presented again for completeness.

13.1.2 p-values for regression coefficients

  • To fit a simple linear regression model, first, use the lm() function, specifying the y-variable and the x-variable, separated by ~ (e.g., lm(y~x)) and saving the information under an object name (e.g., model1). Then use the summary() function on the saved lm object (model1) to obtain the regression coefficient values:
model1<-lm(lhs$FEVFVC02~lhs$f10cigs)
summary(model1)

As a reminder, the main information of interest is in the table under the header Coefficients:.

The regression coefficient values are under the column Estimate, with the intercept denoted in the first row and the slope denoted in the second row. The information in this column was of interest last week.

The standard errors for each of the regression coefficients are under the column Std. Error. This value is used to create confidence intervals and t-test statistics for the coefficients.

In the third column, t value represents the t-test statistics for testing the regression coefficients. It is found by taking the Estimate divided by Std. Error.

And lastly, the p-values for testing the regression coefficients are presented in the column Pr(>|t|). These last three pieces of information are the focus this week.

Additional model statistics that may be of interest include \(R^2\), which is denoted next to Multiple R-squared, and \(\hat{\sigma}\), which is denoted next to Residual standard error. \(R^2\) is a measure of how well the model fits the data and the residual standard error, \(\hat{\sigma}\), is the standard deviation of the residual values.

13.1.3 Confidence intervals for regression coefficients

Besides testing the regression coefficients against the null hypothesis, you can also produce confidence intervals for the regression coefficients to estimate the population value.

  • To obtain confidence intervals for the regression coefficients, use the confint() function on the saved lm object (model1) and specifying the confidence level (level=):
confint(model1, level=0.95)

The output displays the lower and upper limits to the \(95\%\) confidence interval for both the intercept and the slope.

The default confidence level is \(95\%\), but you can modify the value for the level= argument if you want a different confidence level (e.g., 0.90, 0.99).

INFERENCE FOR SIMPLE LINEAR REGRESSION

Purpose: To estimate regression coefficients and their inferential statistics to investigate the relationship between two numerical variables: a response variable, Y, and an explanatory, X.

  • Simple Linear Regression in R [STHDA].

13.3 Model diagnostics

Another step in SLR is to evaluate how well the model fits the data by creating model diagnostics. From these, you assess the assumptions of the linear regression model (via diagnostic plots, such as a residual plot).

13.3.1 Diagnostic plots

    • To obtain diagnostic plots for a multiple linear regression model, use the autoplot() function from the {ggfortify} package on the saved lm object (model1):
#############
# Run the install.packages() function only once per package
############# 
install.packages("ggfortify")

###Access the package using the library() function 
library(ggfortify)

####Obtaining diagnostic plots for the model
autoplot(model1)

As a reminder, four plots will be created in a 2x2 grid. You may need enlarge the Plot window in order to see the four plots clearly.

The first plot (top left; first row, first column) is the Residual plot. It is used to assess the linearity and equal variance assumptions.

The second plot (top right; first row, second column) is the QQ plot of the residuals from the model and is used to assess the normality assumption of the residuals.

The third plot (bottom left; second row, first column) is called the Scale-Location plot and it is a plot of the square root of the standardized residuals vs. the fitted values. This is another plot that can be used to assess the equal variance assumption.

The fourth and final plot (bottom right; second row, second column) is the Residuals vs. Leverage plot and is used to help find influential points, if any.

MODEL DIAGNOSTICS

Purpose: To evaluate the fit and assumptions of the model using plots (e.g., residual plot, QQ plot).


For more information on how to evaluate these plots, check out the following website: Understanding Diagnostic Plots for Linear Regression Analysis.

For additional R resources on model diagnostics, see:

  • Linear Regression Assumptions and Diagnostics in R: Essentials [STHDA].
  • Regression Diagnostics [Boston University].
